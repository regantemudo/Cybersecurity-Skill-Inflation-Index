## 📌 What Is CSII?

Cybersecurity Skill Inflation Index (CSII) measures how job requirements are rising relative to salary and experience.

Components:
- 📅 **Years required**
- 🧠 **Certifications mentioned**
- 🧰 **Tools demanded**
- 📈 **Framework complexity (NIST, SOC 2, ISO 27001, etc.)*

Built to help you:
- Spot inflation before others
- Track long-term skill demand trends
- Prioritize certifications with real data
- Negotiate offers more effectively
