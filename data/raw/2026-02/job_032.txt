Cloud Security Engineer

Bangalore (2 days onsite/week)

Core Skills Required (Top Priority)

    Disaster Recovery (DR) Strategy & Ownership
        RTO/RPO design
        BIA alignment
        DR runbooks & tabletop exercises
        Ransomware resilience planning
        Backup validation & immutable backups
    Vulnerability Management at Scale
        Authenticated scanning (Windows, Linux, Network)
        CVSS interpretation & risk-based prioritization
        Tooling: Prisma, Rapid7, Nessus, etc.
        Triage, remediation coordination, false positive reduction
        Pen-test report analysis (cloud focus)
    Cloud Security Architecture
        Azure / AWS / GCP / AliCloud (multi-cloud)
        IaaS / PaaS / SaaS security controls
        IAM, VNET/VPC, identity flows
        Container security
        CDN / Firewall / Network segmentation review

Technical Expertise

    Hybrid architecture (on-prem + cloud)
    Web & mobile tiered architectures
    Threat modelling fundamentals
    OWASP Top 10
    DNS failover, load balancing, redundancy (active-active / passive)
    Control mapping to NIST, ISO 27001, ISO 22301, SOC 2
    Orchestration of vulnerability tools across multi-cloud

Ideal Profile

    Hands-on cloud security architect
    Strong DR ownership experience
    Enterprise vulnerability governance exposure
    Multi-cloud environment experience
    Able to operate strategically and technically
